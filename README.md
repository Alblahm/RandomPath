## How to use

### Local installation
Clone this repository with [git](https://git-scm.com/) and launch your own [Jupyter Notebook](https://jupyter.org/) server.
```
git clone https://github.com/Alblahm/RandomPath.git
cd RandomPath
jupyter notebook
```
